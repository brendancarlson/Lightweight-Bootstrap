Lightweight-bootstrap

An extremely lightweight bootstrap starter template with offloaded CSS and JS

Download this project as a .zip file Download this project as a tar.gz file

Lightweight Bootstrap is exactly what it sounds like... an extremely lightweight bootstrap starter template.

All you need to use it is a web server. The CSS and JS are even offloaded to the Bootstrap CDN so all you need to serve up is this page.

This page is completely open source, feel free to use it and do whatever the heck you want with it!
